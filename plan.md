## Proyecto: Plataforma Filosofía Interactiva
## Objetivo

Crear una web tipo revista digital para clases de filosofía, inspirada en el estilo visual de filco.es, pero orientada a unidades didácticas.

Público

Estudiantes de secundaria.

Estructura general
Home (estilo revista)

Imagen principal (hero)

Grid de cards por unidad

Hover con leve zoom y sombra

Navegación sticky superior

Diseño responsive

Cada Unidad (8 secciones internas)

Problema inicial

Contexto histórico

Conceptos fundamentales

Mapa argumental

Comparación de posturas

Aplicación contemporánea

Actividad interactiva

Síntesis y reflexión

Requisitos técnicos

HTML + CSS + JS puro (sin frameworks por ahora)

Código limpio y modular

Comentarios explicativos

Fácil de subir a GitHub Pages

Estilo visual

Minimalista

Tipografía clara

Cards con imagen + título + resumen

Efecto hover suave

Inspiración visual en filco.es (no copiar código)
